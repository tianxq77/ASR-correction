
**ASR-Correction**  
语音识别（ASR）校正系统  


一个基于 Python 的自动语音识别（ASR）系统，支持语音转文本及文本后处理校正功能。  

---


**1. 安装依赖**  
```bash
pip install -r requirements.txt
```
- 需要单独安装FFmpeg


**2. 项目结构**  
```
myASR/
├── .idea/               
├── data/                 # 示例数据集
│   ├── testTTS.py        # 音频生成脚本
│   └── std_*.wav         # 示例音频
├── main.py               # 主程序入口
└── README.md            
```
